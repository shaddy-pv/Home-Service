# Appliance Aid Hub

React + Vite + TypeScript web app with Tailwind CSS and shadcn-ui.

## Requirements

- Node.js 18+ and npm

## Getting Started

```sh
# install
npm install

# start dev server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```

## Tech Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- shadcn-ui

## Project Structure

- `src/` app code
- `public/` static assets
- `vite.config.ts` Vite configuration
